Zoomable Tree Map with Jaccard Similarity Clsuters

Inside this zoomable treemap, the initial state shows 17 different Jaccard Similarity clusters. If users click in a certain cluster, then the graph will show 4 attack types’ count inside the current cluster. By clicking in certain attacker type, users will know a more detailed summary based on features in the attack type.
